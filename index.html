Test Your Knowledge

Remember the key points:

  • The img tag is self-closing (no closing tag needed)
  • src attribute is required (tells where the image is)
  • alt attribute is required (describes the image)
  • width and height are optional (no units in HTML attributes)
  • Always use quotes around attribute values
  • Check your file paths and extensions
